# Score Counter
My first attempt at writing React Native app.

![](https://raw.githubusercontent.com/koko57/daGame/master/assets/img/app.gif)

## Built with
React Native, Redux, Styled-Components

## Prerequisities (for Mac)
Node, Watchman
```bash
brew install node
brew install watchman
```
The React Native CLI
```bash
npm install -g react-native-cli
```
XCode and Simulator

## Installation
Download/clone the repo then run

```bash
npm install
```
To run the app:

```bash
# in terminal:
react-native run-ios
```
or open game.xcodeproj in XCode
